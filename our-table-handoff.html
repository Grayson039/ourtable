Developer Notes
Design System

All tokens live in our-table-design-system.js. Import it before any screen file. Exposes the global OT object.

OT.L / OT.D      — Light / Dark theme objects
OT.T         — Typography scale (sz, wt, sans, serif)
OT.S         — Spacing constants (xs → xxl)
OT.R         — Border radius (xs → pill)
OT.c         — Component functions (btn, navBar, statusBar…)
OT.nav()     — Navigation factory (exposes otGo, otBack, otToggleTheme)

// Usage
var inst = OT.nav(screenMap, 'home', 'phone-screen', opts);
window.otGo('recipe');   // navigate to screen
window.otToggleTheme();  // switch light ↔ dark
